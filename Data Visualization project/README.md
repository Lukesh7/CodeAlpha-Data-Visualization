# CodeAlpha Data Science Internship

## Task 3: Data Visualization

### 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Science Internship**.

The objective of this task is to transform raw data into meaningful visualizations that help identify trends, patterns, and insights. Various charts and graphs were created using Python libraries to effectively communicate the information contained in the dataset.

The dataset used for this project was generated through web scraping in **Task 1** and analyzed in **Task 2**.

---

## 🎯 Objectives

- Transform raw data into meaningful visualizations.
- Explore trends and patterns using charts and graphs.
- Build a dashboard for better data interpretation.
- Present insights to support data-driven decision making.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains information about books collected from the **Books to Scrape** website.

### Features

- Title
- Price
- Rating
- Availability

---

## 📊 Visualizations Created

- Price Distribution Histogram
- Book Rating Distribution
- Top 10 Most Expensive Books
- Average Price by Rating
- Book Availability Pie Chart
- Price vs Rating Scatter Plot
- Correlation Heatmap
- Book Price Boxplot
- Data Visualization Dashboard

## 📌 Key Insights

- The dataset contains **1000 books**.
- Most books are priced between **£20 and £40**.
- Books with ratings **3 and 5** appear most frequently.
- Book prices show only a weak relationship with ratings.
- The dataset contains very few price outliers.
- Most books are available in stock.
